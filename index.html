Conoce a Dulce

Soy estudiante de Ingeniería en Sistemas Computacionales con una sólida formación académica en el Instituto Tecnológico de Zitácuaro, y una gran pasión por el desarrollo web. Cuento con experiencia en tecnologías como HTML, CSS, JavaScript, Node.js, React, PHP y el manejo de versiones con Git/GitHub.

Soy una persona orientada al aprendizaje continuo, siempre en busca de mejorar y ampliar mis conocimientos para aportar valor en cada proyecto.

Me motiva trabajar en equipo y colaborar en entornos que desafíen mis habilidades y contribuyan a mi crecimiento profesional.

Front End

Me apasiona el diseño web, especialmente el Frontend, que considero fundamental para captar la atención del usuario. Aspiro a que mis proyectos sean atractivos e intuitivos, brindando una experiencia visual agradable y funcional.

He desarrollado sitios web responsivos aplicando habilidades en maquetación con HTML, interactividad dinámica con JavaScript, y diseño estilizado con CSS y Bootstrap.

Concluí satisfactoriamente el curso de Tecnolochicas Pro, donde adquirí una sólida base en HTML, JavaScript y CSS, fortaleciendo mis habilidades en desarrollo web front-end.

Actualmente, estoy desafiando mis conocimientos al practicar con React, profundizando en la creación de interfaces interactivas y modernas.

HTML CSS Bootstrap JavaScript React

Back End

Tengo experiencia en programación Back End, especialmente en el uso de PHP para desarrollar aplicaciones conectadas a bases de datos MySQL e integradas con servicios SOAP.
También he trabajado con Node.js y MongoDB, utilizando Express junto con herramientas como bcrypt, jsonwebtoken, cors y nodemailer para implementar un sistema de autenticación de usuarios seguro.
Además, tengo experiencia en la gestión de contenedores Docker.
